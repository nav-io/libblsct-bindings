TxIn¶

class blsct.TxIn(obj=None)¶

Represents a transaction input in a confidential transaction.

>>> from blsct import OutPoint, SpendingKey, TokenId, TxId, TxIn, TX_ID_SIZE
>>> import secrets
>>> amount = 123
>>> gamma = 100
>>> spending_key = SpendingKey()
>>> token_id = TokenId()
>>> tx_id = TxId.from_hex(secrets.token_hex(TX_ID_SIZE))
>>> out_point = OutPoint.generate(tx_id, 0)
>>> tx_in = TxIn.generate(amount, gamma, spending_key, token_id, out_point)
>>> tx_in.get_prev_out_hash()
TxId(7b0000000000000064000000000000003ff98b71ff7189fb12d4b93704139753)
>>> tx_in.get_prev_out_n()
37194817
>>> tx_in.get_script_sig()
Script(341a3e3e18b462d20000000000000000000000000000000000000000)
>>> tx_in.get_sequence()
0
>>> tx_in.get_script_witness()
Script(ffffffffffffffff1b585a44e980f30b16ef75db34f7a6d56fe7cee4)
static generate(amount: int, gamma: int, spending_key: SpendingKey, token_id: TokenId, out_point: OutPoint, rbf: bool = False) → Self¶
