ChildKey¶

class blsct.ChildKey(seed: Scalar)¶

Represents a child key. A child key is a Scalar and introduces no new functionality; it serves purely as a semantic alias. BlindingKey, TokenKey and TxKey are exclusively derived from a ChildKey.

>>> from blsct import ChildKey, Scalar
>>> s = Scalar()
>>> k = ChildKey(s)
>>> k.to_blinding_key()
Scalar(610a6f73231d115a54aee8b43c6d6fc5349aa4c45dc9987aad3f7a98fc6249d8)
>>> k.to_token_key()
Scalar(58fd2bdf696268ecfbed529b4968dbfa0e324e5cbe00b18a6398c7feaa9627a4)
>>> k.to_tx_key()
TxKey(2fc0697ce315e42491d60e278ac729802be887e912f536f67a8eea9dc4b2900c)
>>>
to_blinding_key() → Scalar¶

derive a blinding key from the child key

to_token_key() → Scalar¶

derive a token key from the child key

to_tx_key() → TxKey¶

derive a tx key from the child key
