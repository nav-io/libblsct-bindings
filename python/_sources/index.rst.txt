navio_blsct
===========

The ``navio_blsct`` Python binding exposes core functionality from
`navio-core <https://github.com/nav-io/navio-core>`_,
enabling the construction of confidential transactions and the
derivation of required keys in Python.

It provides a Pythonic interface over the underlying C++ library
and is available as a `PyPI package <https://pypi.org/project/navio-blsct/>`_.
