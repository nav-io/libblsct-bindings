PublicKey¶

class blsct.PublicKey(obj: Any = None)¶

Represents an element in the BLS12-381 G1 curve group that is used as a public key.

>>> from blsct import Point, PublicKey, Scalar, ViewKey
>>> s = Scalar.random()
>>> PublicKey.from_scalar(s)
PublicKey(a4a04797481bd531f9cb56...)
>>> p = Point.random()
>>> PublicKey.from_point(p)
PublicKey(b09a14601bee3102a6db45...)
>>> pk = PublicKey.random()
>>> pk.pretty_print()
'1 70896870760eba69c20a1f0d740855a91560a...'
>>> vk = ViewKey()
>>> PublicKey.generate_nonce(pk, vk)
PublicKey(91458dc61b63095b1d5f13...)
>>> pk.get_point()
Point(a70896870760eba69c20a1f0d7...)
f9420ac559ebc)
>>> pk2 = PublicKey.random()
>>> pk == pk2
False
>>> pk == pk
True
>>> ser = pk.serialize()
>>> deser = PublicKey.deserialize(ser)
>>> ser == deser.serialize()
True
