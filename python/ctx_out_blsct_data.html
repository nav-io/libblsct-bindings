CTxOutBlsctData¶

class blsct.CTxOutBlsctData(ctxout_obj: Any = None)¶

Represents a blsct-related data in transaction output in a constructed confidential transaction. Also known as CTxOutBlsctData on the C++ side. This class provides access to the CTxOutBlsctData object, but does not own neither the CTxOut nor CTxOutBlsctData object.

For code examples, see the ctx.py class documentation.
